# 炸弹人-无限

无限关卡版，还原了经典炸弹人的所有功能
Unity版本：2019.4.16f

# 功能说明(技术点)

- 简单AI
- 穿墙校正
- 爆炸位置判断
- 动态地图生成
- 可配置化地图生成
- 其他简单功能（UI控制等什么乱七八糟的）
